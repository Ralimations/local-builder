# Local Builder

Local Builder is a builder-style front-end tool for assembling structured UI blocks, editing layout pieces, and supporting export-oriented workflows from a local interface.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- Puck Editor

## Focus

- Visual layout assembly
- Builder-style editing workflows
- Structured block configuration
- Export and packaging-oriented front-end tooling

## Development

Prerequisites:

- Node.js

Run locally:

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
```

## Notes

This project reflects an interest in no-code or low-code style interfaces, front-end tooling, and configurable UI systems.
